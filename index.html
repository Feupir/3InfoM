Versionamento

O GIT é um software que permite realizar versões de um projeto. O GitHub Desktop é um software que faz a ligação entre, VsCode, Git e GitHub.

O GitHub Desktop permite fazer a sincronia entre o repositório local e o repositório remoto no GitHub.

Commit

Commit é o proceso de confirmação de uma versão

Push

Esse comando envia (sincroniza) as alterações realizadas no repositório local com o repositório remoto no GitHub.
